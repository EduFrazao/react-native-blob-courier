# react-native-blob-downloader

Download blobs

## Installation

```sh
npm install react-native-blob-downloader
```

## Usage

```js
import BlobDownloader from "react-native-blob-downloader";

// ...

const result = await BlobDownloader.multiply(3, 7);
```
